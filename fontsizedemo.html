A browser has a default font size

To change the efault font size:

  • Firefox: Tools -> Options, Content. Change the default font size (FF is 16).
  • Chrome: Settings -> Show Advanced Settings, Web Content: Font Size (default is 16).
  • Internet Explorer: Tools -> Internet Options, Accessibility, "Ignore font styles specified on web page", b) View -> Text size (pick one).

This is just text. Here is an 'm'. Be sure to refresh the page if changing the browser font-size. Chnage the default font size and not which types change (%, em) and which don't (pt, px).
